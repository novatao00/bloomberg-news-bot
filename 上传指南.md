# 🚀 快速上传到GitHub指南

## 第一步：创建GitHub仓库

1. 访问：https://github.com/new
2. 仓库名：**bloomberg-news-bot**
3. 选择 **Public**（免费）
4. **不要**勾选任何选项
5. 点击 **Create repository**

---

## 第二步：上传文件（拖拽方式）

### 打开仓库
创建完成后，点击网页上的 **"uploading an existing file"** 链接

### 拖拽以下4个文件夹（一次性）

```
📁 .github/
📁 github-actions-src/
📁 server/
📁 scripts/
```

### 拖拽以下文档文件

```
📄 README.md
📄 QUICKSTART.md
📄 CHECKLIST.md
📄 UPLOAD_TO_GITHUB.md
📄 .env.example
```

---

## 第三步：提交更改

在页面底部：
- **第一行**输入：Initial commit
- **第二行**输入：Create Bloomberg News Bot
- 点击绿色按钮 **Commit changes**

---

## 第四步：测试GitHub Actions

1. 点击仓库顶部 **Actions** 标签
2. 点击 **Run workflow**
3. 点击绿色的 **Run workflow** 按钮

等待3-5分钟，你应该看到所有步骤变成绿色 ✓

---

- [ ] 所有文件夹已上传
- [ ] GitHub Actions可以运行
- [ ] Artifacts生成成功

## 🎯 上传后下一步

按照 QUICKSTART.md 继续部署到CentOS服务器！
